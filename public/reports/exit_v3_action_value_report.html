Peak20 动作价值卖出模型 V3

这版把卖出问题改成动作价值比较：继续持有价值 - 明日开盘卖出价值。只有模型认为继续持有优势明显偏低，且继续持有胜算也偏低时，才次日开盘卖出。

同时加入市场环境特征，并设置前4个持仓日不轻易卖出，避免过早卖飞20日高收益机会。

动作价值定义

item definition
sell_value 次日开盘可卖出时的净收益
hold_value 未来剩余窗口最佳收盘净收益 - 0.35 × 继续持有相对明日卖出的潜在亏损
q_advantage hold_value - sell_value；越大越应该继续持有，越小越应该卖出
sell rule 持有至少5天，且 pred_q_advantage < 0.050，且 hold_good_prob < 0.55，则次日开盘卖出；跌停不可卖则继续
